# Sound Effects Notice

Please note that the Sonniss GDC Audio Bundle license allows these sound 
effects to be used in commercial games, but strictly prohibits the 
redistribution of the raw audio files within open-source repositories 
or public archives.

Because of this restriction, the Sonniss `.ogg` files are not included 
in this repository. Please see `credits.md` for the exact filenames 
and links to download them from the official sources to place them here.
