# 🍣 Sushi Gakusei

## Game Overview

| Data | Value |
| --- | --- |
| Title | sushi gakusei |
| Genre | Memory / Pattern |
| Platform | PC / Web / Mobile |
| Controls | Keyboard buttons, Touch buttons, Gamepad buttons |
| Session Lenght | 5-15 Minutes |

The player is a student learning from a strict instructor. The instructor gives preparation instructions using **kitchen sounds**. The player must memorize and repeat the sequence correctly to complete sushi orders.

Each successful order leads to a longer or faster sequence.

# Core Gameplay Loop

1. Instructor gives an audio sequence of cooking actions
2. Quick visual icons also appear for accessibility
3. Player repeats the sequence with keyboard inputs
4. If correct: next order
5. If wrong: lose a life
6. Game speeds up gradually

Loop continues until:

- player completes all orders (Arcade)
- player loses all lives
- player quits
